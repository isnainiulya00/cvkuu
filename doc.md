## 1. Deskripsi Aplikasi

Aplikasi ini merupakan Single Page Application (SPA) yang dibangun menggunakan **ReactJS (Vite)** untuk menampilkan **Curriculum Vitae (CV) Mahasiswa**. Aplikasi terdiri dari tiga bagian utama, yaitu:

1. **Profile** â€“ Menampilkan data diri, foto, headline, dan informasi umum mahasiswa.
2. **Skills** â€“ Menampilkan daftar kemampuan teknis/non-teknis dengan icon dan level.
3. **Experiences** â€“ Menampilkan pengalaman organisasi, magang, maupun proyek.

Seluruh data di-load secara otomatis dari file **dataMahasiswa.json**, sehingga komponen tidak di-hardcode.

Aplikasi ini dibuat dengan desain modern, responsif, dan mudah dibaca.

## 2. Teknologi yang Digunakan

* **ReactJS (Vite)**

* JSON sebagai sumber data
